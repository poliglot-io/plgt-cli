List all secrets in a workspace.

Shows secrets defined in matrices with their current value status.

## Synopsis

```bash
plgt secrets list [--workspace <TEXT>] [--prefix <TEXT>]
```

## Options

| Flag | Description |
| --- | --- |
| `--workspace` / `-w` | Target workspace. Uses default workspace if not specified. |
| `--prefix` / `-p` | Filter secrets by URI prefix. |
