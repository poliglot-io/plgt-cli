Uninstall a package from your workspace.

## Synopsis

```bash
plgt uninstall [package_name <TEXT>] [--workspace <TEXT>] [--yes]
```

## Options

| Flag | Description |
| --- | --- |
| `package_name` | The package name to uninstall. |
| `--workspace` / `-w` | Target workspace. Uses default workspace if not specified. |
| `--yes` / `-y` | Skip confirmation prompt. |
