---
title: "plgt secrets set"
description: "Set a secret's value."
---

Set a secret's value.

By default prompts for the value interactively (masked input).
Use --from-stdin to pipe the value.

## Synopsis

```bash
plgt secrets set [secret_id <TEXT>] [--workspace <TEXT>] [--from-stdin]
```

## Options

| Flag | Description |
| --- | --- |
| `secret_id` | The secret ID (e.g., matrix:SecretName). |
| `--workspace` / `-w` | Target workspace. Uses default workspace if not specified. |
| `--from-stdin` | Read value from stdin instead of interactive prompt. |
