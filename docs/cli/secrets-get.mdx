Get a secret's metadata or value.

By default shows metadata. Use --value to retrieve the decrypted value.

## Synopsis

```bash
plgt secrets get [secret_id <TEXT>] [--workspace <TEXT>] [--value]
```

## Options

| Flag | Description |
| --- | --- |
| `secret_id` | The secret ID (e.g., matrix:SecretName). |
| `--workspace` / `-w` | Target workspace. Uses default workspace if not specified. |
| `--value` / `-v` | Retrieve and display the decrypted secret value. |
