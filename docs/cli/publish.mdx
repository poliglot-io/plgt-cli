Build the package and publish it to the registry under the caller's publisher.

## Synopsis

```bash
plgt publish [--config <PATH>] [--dry-run]
```

## Options

| Flag | Description |
| --- | --- |
| `--config` / `-c` | Path to poliglot.yml. Defaults to poliglot.yml in the current directory. |
| `--dry-run` | Run all validation server-side without uploading or persisting. |
