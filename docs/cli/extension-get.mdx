Get a single extension by ID.

If --output is specified, writes the Turtle content to the file.
Otherwise, displays extension details and content.

## Synopsis

```bash
plgt extension get [extension_id <TEXT>] [--workspace <TEXT>] [--output <PATH>]
```

## Options

| Flag | Description |
| --- | --- |
| `extension_id` | The extension ID. |
| `--workspace` / `-w` | Target workspace. Uses default workspace if not specified. |
| `--output` / `-o` | Output file path for the Turtle content. |
