Scaffold a new migration file.

## Synopsis

```bash
plgt migration new [--from <TEXT>] [--matrix <TEXT>]
```

## Options

| Flag | Description |
| --- | --- |
| `--from` | Source version that the migration transforms data away from. |
| `--matrix` | Matrix name (required when poliglot.yml declares more than one). |
