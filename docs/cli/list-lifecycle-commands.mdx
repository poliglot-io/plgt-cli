List lifecycle commands for a package.

## Synopsis

```bash
plgt list-lifecycle-commands [package_name <TEXT>] [--workspace <TEXT>] [--limit <INTEGER>]
```

## Options

| Flag | Description |
| --- | --- |
| `package_name` | The package name to list lifecycle commands for. |
| `--workspace` / `-w` | Target workspace. Uses default workspace if not specified. |
| `--limit` / `-n` | Maximum number of commands to show. Default: `10`. |
