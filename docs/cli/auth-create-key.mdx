---

Create a new personal access token.

## Synopsis

```bash
plgt auth create-key [name <TEXT>] [--expires <FUNCTION>]
```

## Options

| Flag | Description |
| --- | --- |
| `name` | Human-readable name for the API key |
| `--expires` | Expiration duration: 7d, 14d, 30d, 90d, or never |
